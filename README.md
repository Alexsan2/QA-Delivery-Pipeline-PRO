# 🚀 QA-Delivery-Pipeline-PRO

Este projeto é um ecossistema de testes automatizados focado em **Continuous Delivery (CD)** e **Continuous Integration (CI)**. Utilizamos as melhores práticas de mercado para garantir uma entrega de software resiliente e segura.

## ⚙️ Arquitetura e Ferramentas
* **Framework:** Selenium WebDriver + Node.js
* **Pattern:** Page Object Model (POM)
* **Gestão Ágil:** Kanban & Scrum (Backlog, Sprints, Dailies)
* **Pipeline:** GitHub Actions (Linting, Test Execution, Artifacts)

## 📊 Fluxo de Trabalho (Kanban)
1. **Backlog:** Definição de requisitos e User Stories.
2. **To Do:** Tarefas priorizadas para a Sprint.
3. **Doing:** Desenvolvimento da automação e scripts.
4. **Testing:** Validação local e em ambiente de Stage.
5. **Done:** Código aprovado pela Pipeline de CI e mergeado em Main.

## 🚀 Como Executar a Esteira
1. `npm install` (Instala as dependências)
2. `npm test` (Executa a regressão local)
